Re: 16X2 LCD+KEYPAD KIT &python3

by adafruit_support_mike on Tue May 27, 2014 6:13 am

The code we've written to support the MCP230xx I2C breakouts uses the 'smbus' module, and I don't think that's been ported to Python3 yet.

It's technically possible to make the hardware work with Python3, but you'd have to find/build an I2C layer and rewrite the library on top of that.

found this here https://github.com/dalgibbard/moonlighter

wget http://ftp.de.debian.org/debian/pool/ma ... ig.tar.bz2
tar xf i2c-tools_3.1.0.orig.tar.bz2
rm -f i2c-tools_3.1.0.orig.tar.bz2
cd i2c-tools-3.1.0/py-smbus
# patch over smbusmodule.c
mv ../../pysmbus-patch/smbusmodule.c .
mv ../../pysmbus-patch/i2c-dev.h .
python3 setup.py build
sudo python3 setup.py install

return "%s"%(datetime.datetime.now().strftime("%Y-%m-%d %H:%M:%S"))

# initialize pygame
print("main")
